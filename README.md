# ElderGuard AI

Elegant React frontend for a mock AI-driven elderly loneliness detection system.

## Features

- Pastel glassmorphism UI with soft gradients
- Role-based login flow for NGO/Caregiver and Family members
- NGO dashboard with hero metrics and feature cards
- Assessment form with fake AI risk evaluation
- Family panel with trend placeholder, summary, and alerts
- Responsive mobile + desktop design

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite, usually `http://localhost:4173`.
